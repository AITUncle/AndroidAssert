android-assert是一个非常简单轻量的android断言库。类似于junit的Assert类。

android-assert不是用来写测试用例的，我们可以直接在项目代码中使用他。在debug模式下，断言失败将会抛出断言异常AssertionFailedError，在release模式下，将不会抛出异常。
